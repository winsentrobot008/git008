# Python 多功能计算器 - 完整代码

我编写了一个完整的Python计算器程序，包含**三种使用模式**，支持加减乘除四则运算：

## 1️⃣ 基础计算器（菜单选择模式）
- 显示菜单让用户选择运算类型（加减乘除）
- 输入两个数字后计算结果
- 支持整数和小数运算
- 包含错误处理（除零错误、无效输入等）
- 可连续计算

## 2️⃣ 进阶计算器（表达式输入模式）
- 直接输入完整表达式，如 `3 + 5`
- 格式：`数字 运算符 数字`
- 同样支持所有四则运算

## 3️⃣ 图形界面计算器（GUI模式）
- 使用 tkinter 构建的桌面图形界面
- 按钮布局类似标准计算器
- 支持鼠标点击输入数字和运算符
- 实时显示计算结果
- 错误提示弹窗

- ✅ 安全处理除零错误
- ✅ 自动识别整数/小数显示格式
- ✅ 良好的用户交互体验
- ✅ 完善的错误处理机制
- ✅ 三种模式可选，灵活使用
