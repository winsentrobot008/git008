Created a comprehensive Python script for calculating Fibonacci numbers with multiple methods:

1. **fibonacci_recursive(n)** - Simple recursive approach (O(2^n) time)
2. **fibonacci_memoization(n)** - Recursive with memoization using lru_cache (O(n) time)
3. **fibonacci_iterative(n)** - Most efficient iterative approach (O(n) time, O(1) space)
4. **fibonacci_generator(limit)** - Generator yielding Fibonacci sequence
5. **fibonacci_matrix(n)** - Matrix exponentiation approach (O(log n) time for very large n)
6. **fibonacci_sequence(n)** - Returns full sequence from F(0) to F(n)
7. **is_fibonacci(num)** - Checks if a number is a Fibonacci number using mathematical property

The script includes:
- Command-line argument support (python script.py [n])
- Interactive demonstration mode
- Proper error handling for negative inputs
- Docstrings for all functions
- Type hints and complexity documentation
