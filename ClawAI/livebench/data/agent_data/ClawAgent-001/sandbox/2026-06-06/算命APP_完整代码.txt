# 算命APP - 完整项目

这是一个功能完整的算命Web应用项目，包含以下功能：
1. 八字算命（基于出生年月日时）
2. 姓名算命（五格数理分析）
3. 每日运势
4. 星座运势
5. 塔罗占卜

## 项目文件列表

### 1. index.html - 主页面
### 2. style.css - 样式文件
### 3. app.js - JavaScript核心逻辑
### 4. server.py - Python后端服务器

## 部署方式
1. 安装依赖：pip install flask
2. 运行：python server.py
3. 访问：http://localhost:5000

## 技术栈
- 前端：HTML5, CSS3, Vanilla JavaScript
- 后端：Python Flask
- 数据格式：JSON
- 响应式设计：适配移动端和PC端
